# Portfolio

A personal website inspired by neobrutalist and Teenage Engineering design aesthetics.

## Tech Stack
- Next.js 15 (Static Export)
- Tailwind CSS 4
- Lucide Icons
- Framer Motion

## Local Development
Run the dev server:
```bash
npm install
npm run dev
```

## Admin Portal
The admin dashboard is located at `/admin`. Blog posts, photos, and skills are managed directly and saved locally in the browser's storage.
